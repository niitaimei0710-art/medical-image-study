# フォルダ展開

## 概要

本プログラムは、指定したフォルダ内のDICOMデータを再帰的に探索し、Series情報を一覧表示するためのプログラムです。

各SEフォルダから代表となるDICOMファイルを読み込み、Series NumberおよびSeries Descriptionを取得します。

DICOMデータの構成確認や、画像解析前のシリーズ整理に利用できます。

---

## 対応環境

- Python 3.10以上

---

## 使用ライブラリ

```
pydicom
```

インストール

```bash
pip install pydicom
```

---

## フォルダ構成例

```
選択フォルダ
│
├── EX1
│   ├── SE1
│   ├── SE2
│   ├── SE3
│   └── …
│
├── EX2
│   ├── SE1
│   ├── SE2
│   └── …
│
└── EX3
```

---

## 使用目的

本プログラムは、MRI装置から出力されたDICOMデータのシリーズ構成を確認するために作成しました。

メーカーごとのSeries構成の確認や、画像解析プログラム作成前のデータ整理に利用できます。

---

## 注意事項

本プログラムはDICOMヘッダ内のSeries NumberおよびSeries Descriptionを取得します。

画像データ（Pixel Data）は読み込まないため、高速にシリーズ情報のみを確認できます。
